this is the first react project in the Odin project curriculum !

## project requirments:
This project main focuse and goal is the utilization and familiarization of react specific technologies and concepts including:
    - vite (react project constructur/ bundeling)
    - JSX
    - react dev tools
    - modular designe
    - component based approach
    - state handelling
    - component props
    - good state structuring 

### user  input:
- general information section:
    * name
    * email
    * phone-number

-  educational experience section :
    * school name
    * title of study
    * date of study

- practical experience section :
    * company name
    * position title
    * main responsibilities of job
    * date fropm and until when you worked

### application functionality:
 - edit and submit button per section OR for the whole CV;
    * (submitBtn) submits and displaysinput field value in HTML elements
    * (editBtn) displays back input fields with previous information as values
                ability to edit and resubmit the content
